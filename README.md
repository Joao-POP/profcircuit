# What is profcircuit?

Profcircuit is a collection of tools to aid electrical engineers (or students) in circuit design and reverse engineering.
It has tools for:
- Simple (ideal) circuit simulation
- Advanced simulation of circuit behavior, from temperature to EMI
- Component and trace layout on a PCB
- Prettifying circuit schematics
- Image analysis for circuit reconstruction

## Features

- Runs on any platform, as long as there is a frontend
- Fully offline
- Works with open standards
